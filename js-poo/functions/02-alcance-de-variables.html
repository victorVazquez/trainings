Alcance de variables (Scope of Variables)

Las variables en Javascript no son definidas en un bloque de código, sino en el ámbito de una función.
Esto significa que si una variable es definida dentro de una función, no es visible fuera de dicha función en que se definio.

Sin embargo, una variable definida dentro de un bloque if o for, por ejemplo, es visible fuera del bloque de código.

La expresión "Variables globales", describe variables que son definidas fuera de cualquier función. Lo cual es opuesto a "Variables locales", que se refiere a toda variable definida dentro de cualquier función.

El código dentro de una función tiene acceso a todas las variables locales, así como a todas sus variables locales.
